# Spotifacts

A simple static website to visualize Spotify listening data. Runs completely on the client side, no data is ever sent to a server, 100% private.

## Features
- Parses Spotify data JSON exports.
- Visualizes listening history and other metrics locally in the browser.

## Setup
1. Request your data from Spotify Privacy settings.
2. Open https://faust504.github.io/spotifacts/ in your browser.
3. Upload your data files when prompted.
